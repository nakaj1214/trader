# ブラウザ検証リファレンス

フロントエンド問題の証拠収集に使用するブラウザ検証手段。
verify-before-fix スキルの Phase 3 で、フロントエンドレイヤーの検証時に参照する。

---

## antigravity IDE のブラウザエージェント

antigravity IDE（Google DeepMind）は内蔵ブラウザエージェントを持つ。
Claude Code と同じ IDE 内で動作するため、ファイル経由で連携可能。

### 利用可能な操作

| 操作 | 説明 |
|------|------|
| スクリーンショット | UI の現在の状態を視覚的に確認 |
| DOM キャプチャ | 実際の HTML 構造・属性・computed style を取得 |
| コンソールログ読み取り | JS エラー、イベント発火順序、デバッグ出力の確認 |
| クリック・入力・スクロール | ユーザー操作の再現、イベント発火テスト |
| 動画撮影 | 操作フロー全体の記録・確認 |

### ファイル連携プロトコル

Claude Code → antigravity エージェントへの検証依頼:

1. Claude Code が `tasks/browser-verify-request.md` に依頼を書く
2. antigravity エージェントがブラウザ操作を実行
3. 結果を `tasks/browser-verify-result.md` + スクリーンショット等に書き出す
4. Claude Code が結果を読み取る

```markdown
<!-- tasks/browser-verify-request.md -->
# ブラウザ検証依頼

## URL
http://localhost:28001/stock

## 検証内容
1. ページを開き、スクリーンショットを撮る
2. `td.editable` をダブルクリックする
3. ダブルクリック後のコンソールログを取得する
4. 該当セルの DOM キャプチャを取得する

## 出力先
tasks/verify-output/
```

## ユーザーへの検証依頼（フォールバック）

antigravity が使えない場合、ユーザーに検証を依頼する:

```markdown
## 確認依頼

現象の原因を特定するため、以下の確認をお願いします:

1. **確認手順**: [具体的な操作手順]
2. **確認ポイント**: [何を見ればよいか]
3. **期待値**: [正常ならこう見える]
4. **異常値**: [問題があればこう見える]

結果を教えていただければ、正確な修正を提案できます。
```
