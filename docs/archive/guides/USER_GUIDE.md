# AI Stock Predictor ユーザガイド

> **はじめに**: このガイドは **株の知識がまったくない方** でも安心して使えるように書かれています。
> 専門用語にはその都度カンタンな説明をつけています。

---

## このツールは何をするの？

AI Stock Predictor は、**アメリカの有名企業の株が来週上がりそうかどうかを AI が自動で予測してくれるツール** です。

毎週日曜日に自動で動き、結果を Slack（チャットツール）に通知してくれます。

```
毎週日曜 9:00 (日本時間)
  ↓
① 約600社の中から「伸びそうな会社」トップ10を自動選出
  ↓
② AIが「来週いくらになるか」を予測
  ↓
③ 結果を Google スプレッドシート（ネット上の表計算）に記録
  ↓
④ 先週の予測が当たったかチェック（的中率を計算）
  ↓
⑤ Slackに結果を通知
```

---

## 株ってそもそも何？

### 株（株式）とは

会社が資金を集めるために発行する「権利書」のようなものです。
株を買う＝その会社の一部を持つオーナーになる、というイメージです。

### 株価が上がる・下がるとは

- **上がる**: その会社の人気が高まる → 買いたい人が増える → 値段が上がる
- **下がる**: その会社に不安がある → 売りたい人が増える → 値段が下がる

### 株で利益を出す基本的な仕組み

```
安いときに買って → 高くなったら売る → 差額が利益
```

例: 100ドルで買った株が 110ドルになったら、10ドルの利益です。

## このツールで使われている指標のカンタン解説

AI が銘柄を選ぶときに見ている4つのポイントです。

| 指標 | 一言でいうと | もう少し詳しく |
|------|------------|--------------|
| **RSI** | 買われすぎ・売られすぎの度合い | 0〜100の数値。70以上＝買われすぎ（そろそろ下がるかも）、30以下＝売られすぎ（そろそろ上がるかも） |
| **MACD** | 上昇トレンドか下降トレンドか | 短期と長期の値動きの差を見る指標。上向きなら上昇の勢いあり |
| **出来高** | どれだけ注目されているか | 売買された株の数。多い＝多くの人が注目している |
| **株価変動率** | 最近の値動きの大きさ | 直近1ヶ月でどれくらい上がったか（%） |

### その他よく出てくる用語

| 用語 | 意味 |
|------|------|
| **ティッカー** | 株の略称コード。例: Apple → AAPL、Google → GOOGL |
| **S&P500** | アメリカの代表的な大企業500社の株価指標。日本でいう「日経平均」のようなもの |
| **NASDAQ100** | テクノロジー企業が多いアメリカの株価指標。Apple, Google, Amazon などが含まれる |
| **時価総額** | 会社全体の値段。「株価 × 発行株数」で計算される |
| **予測上昇率** | AIが予測する来週の値上がり幅（%）|
| **信頼区間** | 予測のブレ幅。小さいほど予測の精度が高い |
| **的中率** | AIの予測が実際に当たった割合 |

## 通知の見方

毎週日曜日に Slack に届く通知の読み方です。

```
📊 週間AI株式予測レポート (2026-02-22)

【今週の上昇予測銘柄】
  1. AAPL: $250.00 → $265.00 (予測 +6.0%, 信頼区間 ±3.2%)
  2. MSFT: $420.00 → $438.00 (予測 +4.3%, 信頼区間 ±2.8%)

【先週の的中率】(判定: 最終営業日終値基準)
  的中: 6/8銘柄 (75.0%)
  累計的中率: 71.2% (過去12週)
```

### 通知の読み方

- **AAPL: $250.00 → $265.00**
  → Apple の株が今 250ドルで、来週 265ドルになると AI が予測
- **予測 +6.0%**
  → 6%上がる予測
- **信頼区間 ±3.2%**
  → 予測のブレ幅。実際は +2.8%〜+9.2% の範囲になる可能性がある
- **的中: 6/8銘柄 (75.0%)**
  → 先週の予測8件中6件が的中（上がると予測して実際に上がった）

## 使い方の流れ

### ステップ1: まず「見るだけ」から始める（1〜2ヶ月）

**最初は絶対にお金を使わないでください。**

```
毎週日曜に Slack 通知が来る
  ↓
「AAPL: $250 → $265 予測」を眺める
  ↓
翌週、Yahoo Finance などで実際の株価を確認する
  ↓
的中率が Slack に自動で届く → AIの精度を観察する
```

的中率が安定して **60〜70% 以上** を維持するかどうか、まず観察しましょう。

### ステップ2: 仕組みを理解する

「なぜこの銘柄が選ばれたのか」を毎週1銘柄だけ調べる習慣をつけると、株の知識が自然につきます。

おすすめの確認方法:
- [Yahoo Finance](https://finance.yahoo.com/) でティッカーを検索
- 会社名・事業内容・最近のニュースをざっと読む

### ステップ3: 少額で試す（慣れてきたら）

- **最初は1銘柄・少額**（数万円程度）から
- 予測通りに買っても、**必ず損切りライン（例: -5%で売る）を決めておく**
- このシステムは「上昇予測銘柄のリスト」であり、**必ず上がる保証はない**

## 株取引を始めるには（日本在住の方）

このツールは **米国株（S&P500・NASDAQ100）** が対象です。
米国株に対応した日本の証券会社で口座を開設する必要があります。

| 証券会社 | 特徴 | おすすめ度 |
|---------|------|----------|
| **SBI証券** | 口座数No.1、情報が豊富 | ⭐⭐⭐ |
| **楽天証券** | 楽天ポイント連携、画面が見やすい | ⭐⭐⭐ |
| **マネックス証券** | 米国株の銘柄数が多い | ⭐⭐ |

口座開設・維持費はどこも **無料** です。初心者には **SBI証券** か **楽天証券** がおすすめです。

## よくある質問（FAQ）

### Q: このツールを使えば必ず儲かりますか？

**いいえ。** AI の予測は外れることもあります。的中率50%以下が続くこともあります。
このツールは「参考情報」を提供するものであり、投資を推奨するものではありません。

### Q: いくらからはじめられますか？

米国株は **1株から** 買えます。1株あたり数十ドル〜数百ドル（数千円〜数万円）程度です。
PayPay証券なら1,000円から購入できます（ただし手数料が割高です）。

### Q: 通知が来ないのですが？

以下を確認してください:
1. Slack のチャンネル `#stock-alerts` に参加しているか
2. Slack の通知設定がオンになっているか
3. GitHub Actions が正常に動いているか（リポジトリの Actions タブで確認）

### Q: 予測の対象はアメリカの株だけですか？

現在は **S&P500**（米国大企業500社）と **NASDAQ100**（米国テック企業100社）が対象です。
設定ファイルで日経225（日本株）を有効にすることもできますが、デフォルトでは無効です。

### Q: 用語がわからないときは？

通知に「用語メモ」が自動で付きます（初心者モードが有効な場合）。
さらに詳しく知りたい場合は、以下のコマンドで用語を検索できます:

```bash
python -m src.glossary RSI
```

引数なしで実行すると、登録済みの全用語一覧が表示されます。

## ⚠️ 重要な注意事項

```
・このツールはAIによる株価予測の学習・実験ツールです
・投資助言を提供するものではありません
・AIの予測が外れることは日常的にあります
・「予測が当たった＝儲かる」ではありません（手数料・為替・タイミングも関係します）
・投資は必ず余剰資金（なくなっても生活に困らないお金）で行ってください
・まずは2〜3ヶ月間、お金を動かさずに的中率を観察するのが最も安全です
```
