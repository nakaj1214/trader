## このプロジェクトの運用の進め方

### ステップ1: まず「見るだけ」から始める（1〜2ヶ月）

実際にお金を動かす前に、システムが出す予測を眺めて感覚をつかみます。

```
毎週日曜に Slack 通知が来る
  ↓
「AAPL: $230 → $238 予測」という銘柄リストを見る
  ↓
翌週、実際の株価がどう動いたか確認する（Yahoo Finance など）
  ↓
的中率が Slack に自動で届く
```

このプロジェクトの的中率が安定して **60〜70% 以上** を維持するか、まず観察してください。

---

### ステップ2: 仕組みを理解する

このシステムが見ているのは以下の4指標です。

| 指標 | 意味（簡単に） |
|------|--------------|
| RSI | 買われすぎ・売られすぎの度合い |
| MACD | 上昇トレンドか下降トレンドか |
| 出来高トレンド | 注目が集まっているか |
| 直近1ヶ月株価変動 | 最近上がり続けているか |

「なぜこの銘柄が選ばれたのか」を毎週1銘柄だけ調べる習慣をつけると、株の知識が自然につきます。

---

### ステップ3: 少額で試す（慣れてきたら）

- **最初は1銘柄・少額**（数万円程度）から
- 予測通りに買っても、**必ず損切りライン（例: -5%）を決めておく**
- このシステムは「上昇予測銘柄のリスト」であり、**必ず上がる保証はない**

---

## 株取引アプリ（日本在住の場合）

このプロジェクトは **米国株（S&P500・NASDAQ100）** が対象なので、米国株に対応した証券会社を選ぶ必要があります。

| 証券会社 | 特徴 | 米国株手数料 |
|---------|------|------------|
| **SBI証券** | 口座数No.1、情報量が多い | 約定代金の0.495%（最低0ドル） |
| **楽天証券** | 楽天ポイント連携、UIが見やすい | 同上 |
| **マネックス証券** | 米国株の銘柄数が多い、ツールが充実 | 同上 |
| **PayPay証券** | スマホ完結、1000円から買える | 売買スプレッドに含む（割高） |

**初心者には SBI証券か楽天証券** が無難です。どちらも口座開設・維持費は無料で、米国株の情報も豊富です。

## 重要な注意点

```
このシステムはあくまで「候補リスト」を出すツールです。

・AIの予測が外れることは普通にある
・的中率50%以下が続いても驚かないこと
・「予測が当たった＝儲かる」ではない（タイミングや手数料も関係する）
・余剰資金（なくなっても生活に困らないお金）だけで行うこと
```

まずは**2〜3ヶ月間、お金を動かさずに的中率を観察する**のが最も安全な始め方です。
