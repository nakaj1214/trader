## Q9. NISAをやったほうが良いのでは？特定口座で収益の20%は取られる？

**A: 両方正しいです。ただしこのシステムとNISAの相性は良くありません。**

### 収益の20.315%については

はい、特定口座で利益が出た場合は **20.315%が自動的に差し引かれます**。
例: 10万円の利益 → 約2万円が税金として引かれ、手取り約8万円

### NISAとの比較

| | **NISA（新NISA成長投資枠）** | **特定口座（源泉徴収あり）** |
|--|--|--|
| 税率 | **0%（非課税）** | 20.315% |
| 年間投資上限 | 240万円 | 上限なし |
| 損益通算 | **できない** | できる |
| 売却後の枠の回復 | 翌年（当年中は使えない） | 制限なし |
| 向いている使い方 | 長期積み立て（数年〜） | 短期売買・高頻度取引 |

### このAIシステムにNISAが向かない理由

このシステムは **週次で売買を繰り返す短期運用** です。NISAで短期売買をすると：

1. **枠が消費され続ける**: 毎週売買すると240万円の年間枠がすぐなくなる
2. **損益通算ができない**: NISA内で損が出ても、他の口座の利益と相殺できない
3. **長期保有の恩恵が薄い**: NISAは数年〜十数年の複利効果を前提に設計されている

### 推奨する使い分け

```
このAIシステム → 特定口座（源泉徴収あり）で運用
NISA（つみたて投資枠/成長投資枠） → 全世界株・S&P500インデックスファンドの長期積み立てに使う
```

**結論:** NISAは「長期インデックス投資」に使い、このシステムは「特定口座での短期AI売買」に使う。
両方を並行して運用するのが最も合理的な資産配分です。

## Q1. 日本株が無効になっている理由

**A:** このシステムは **米国株（S&P500・NASDAQ100）用** として構築されています。

技術的な理由：
- データソース（yfinance）は米国株を主軸に設計されており、日本株のデータ精度が劣る場合がある
- 日本市場は取引時間・祝日カレンダー・通貨（円）が異なるため、現在のコードをそのまま適用できない
- 単元株制度（最低100株単位）への対応が別途必要

有効化する場合は `config.yaml` を編集します:
```yaml
markets:
  - "sp500"
  - "nasdaq100"
  - "nikkei225"   # ← この行のコメントを外す
```
ただし動作確認が必要です。まず米国株だけで運用に慣れることを推奨します。

## Q2. なぜ米国株から始めるのか

**A:** 主に3つの理由があります。

| 理由 | 説明 |
|------|------|
| データの質 | 米国株はyfinanceのデータが豊富・正確 |
| 流動性 | S&P500銘柄は出来高が大きく価格操作が起きにくい |
| 1株から買える | 分数株（フラクショナルシェア）対応の証券会社なら少額から始められる |

日本株はいずれ追加可能ですが、**まず1つの市場で運用方法を習得する**のが初心者には合理的です。

## Q3. 米国株と日本株を両方やらないのか（リスク分散では？）

**A:** 「分散」の考え方は正しいですが、**始めの段階では逆効果になる可能性があります**。

- 2つの市場を同時に管理すると確認・判断の負担が2倍になる
- 為替リスク（円安/円高）が加わり、損益計算が複雑になる
- 日本株用の設定・テストが追加で必要で、バグが混入しやすい

**推奨するアプローチ:**
```
Step 1: 米国株のみで6〜12ヶ月運用 → システムの動作と的中率を確認
Step 2: 安定したら日本株を有効化して並行運用
```

ある程度慣れてから日本株を追加する方が、リスクを把握しながら拡張できます。

## Q4. 社会人だが株の利益は確定申告が必要か

**A:** **口座の種類によって変わります。** 以下の表で確認してください。

| 口座の種類 | 確定申告 | 会社への影響 |
|-----------|---------|------------|
| **特定口座（源泉徴収あり）** | **不要**（証券会社が自動処理） | なし |
| 特定口座（源泉徴収なし） | 利益が20万円超なら必要 | 住民税増加で会社に分かる可能性あり |
| 一般口座 | 原則必要 | 住民税増加で会社に分かる可能性あり |
| **NISA口座（つみたてNISA含む）** | **不要**（非課税） | なし |

**初心者への推奨:** 「**特定口座（源泉徴収あり）**」を選ぶ。証券会社が税金を自動で引いてくれるため確定申告不要で、会社の給与と切り離して管理できます。

税率は利益の **20.315%**（国税15.315% + 地方税5%）です。

> 米国株の場合、配当金は米国で10%源泉徴収されます。「外国税額控除」の確定申告で一部取り戻せますが、少額の場合は手間に見合わないことも多いです。

## Q5. 日本株の最低投資額がなぜ米国株より高いのか

**A:** 日本株には **単元株制度** があるためです。

- 日本株は原則 **100株単位** でしか売買できない
- 株価が 3,000円の銘柄 → 1単元 = 3,000円 × 100株 = **30万円**
- 株価が 500円の銘柄 → 1単元 = 500円 × 100株 = **5万円**

一方、米国株は **1株単位（または分数株）** で買えます。

| 比較 | 米国株 | 日本株 |
|------|-------|-------|
| 最小単位 | 1株（〜分数株） | 100株 |
| $100（約15,000円）での購入 | できる（1株単位） | できない（単元に届かない） |
| 少額分散 | しやすい | しにくい |

これが「日本株で分散しようとすると50〜100万円必要」という理由です。

## Q6. $500〜$1,000は少額扱いなの？

**A:** 「少額」の感覚は人によって違いますが、**投資の文脈では$500〜$1,000は入門的な金額**です。

わかりやすく円換算すると（1ドル＝150円として）：

| 金額 | 円換算 | 何に相当するか |
|------|--------|-------------|
| $500 | 約7.5万円 | 家電1台分程度 |
| $1,000 | 約15万円 | 入門レベルの投資資金 |
| $5,000 | 約75万円 | ある程度分散できる金額 |
| $10,000 | 約150万円 | 本格的な分散投資が可能 |

**$500〜$1,000を「少額」と呼ぶ理由:**

- このシステムは最大10銘柄を選出 → 均等に分散すると1銘柄あたり $50〜$100
- 取引手数料（0〜数百円）を考えると、1銘柄 $50 では手数料負けのリスクがある
- 「少額」＝「リスクを最小に抑えながら仕組みを学ぶための金額」という意味

**初心者として安心な出発点は「万が一全損しても生活に影響しない金額」です。**

## Q8. 個人の財務状況に基づく投資額の提案

**あなたの状況:**
- 社会人2年目、年収370万、手取り月21万
- 毎月貯金5万、現在の貯金180万

### 緊急予備資金の確保（最優先）

投資を始める前に、**生活費6ヶ月分を確保**してください。

```
月の生活費 ≈ 手取り21万 − 貯金5万 = 16万円（生活費）
緊急予備資金 = 16万 × 6ヶ月 = 96万円
```

| 資金区分 | 金額 | 用途 |
|---------|------|------|
| 緊急予備資金 | **96万円** | 絶対に投資しない（失業・病気への備え） |
| 投資可能額 | **84万円** | 180万 − 96万 = 残り |
| 実際に使う初期投資 | **20〜30万円** | 投資可能額の1/3程度から開始 |

### 具体的な投資プラン

**初期投資（3〜6ヶ月後）:**
```
まず $1,000〜$2,000（約15〜30万円）で実際の投資を開始
→ 予測精度が確認できるまでこれ以上は増やさない
```

**毎月の積み立て:**
```
貯金5万円のうち 2万円を投資に回す（残り3万は貯蓄継続）
→ 半年で 12万円（$800程度）の追加投資
→ 的中率が確認できたら増額を検討
```

**1年後の目標資産配分:**
```
緊急予備資金: 96万円（固定）
投資中: 30〜50万円
貯蓄継続: 残り（新規貯金も継続）
```

### 注意点

- **最初の3ヶ月は一切投資しない**（データ蓄積期間）
- 投資に回す金額は「なくなっても生活できる額」に限定する
- 毎月2万を積み立てても年間24万円。焦らず長期で見る
- まずは **NISA口座（つみたてNISA/新NISA）** でインデックス投資との比較も検討する価値あり

> **参考:** 同じ20〜30万円を新NISA（全世界株インデックス）に入れた場合、年率5〜7%で運用できると仮定すると、リスクはこのAIシステムよりはるかに小さい。両方試してパフォーマンスを比較するのも有益です。
